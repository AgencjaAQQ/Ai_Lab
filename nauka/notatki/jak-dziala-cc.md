# Jak działa Claude Code

## Komendy
- /help — lista komend i skrótów
- /status — info o sesji, modelu, folderze
- /compact — kompresuje kontekst (czyści biurko, zostawia notatki)
- /clear — totalny reset rozmowy

## Zasady
- CC widzi tylko folder, w którym go odpaliłeś
- CC czyta pliki na żądanie, nie wszystkie na raz
- Kontekst (pamięć robocza) się zapełnia — /compact pomaga
- CC pyta o zgodę przed edycją plików
- Do CC piszesz po polsku, komendy ze slashem po angielsku
