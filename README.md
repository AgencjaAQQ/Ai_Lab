# AI Lab — Piotr Gajda

Osobiste repozytorium łączące naukę AI/Claude Code z pracą nad projektami klienckimi.

## O repozytorium

Centrum nauki i eksperymentów z Claude Code, przepływami pracy opartymi na AI oraz narzędziami developerskimi. Równolegle — miejsce przechowywania skryptów i plików roboczych z projektów klienckich.

## Struktura

```
ai-lab/
├── nauka/              # Kurs Claude Code: dziennik nauki, notatki, dokumentacja
│   └── notatki/        # Pliki tematyczne (jak działa CC, itp.)
├── projekty/           # Projekty klienckie
│   └── betondur/       # Skrypty i dane — klient Betondur
└── narzedzia/          # Narzędzia pomocnicze, konfiguracje
```
